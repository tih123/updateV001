MVC TEMPLATE INSTRUCTIONS

1 - Copy Content/StateTemplate folder to your project Content folder

2 - Copy desired cshtml files from Views/Shared/modules to your Views/Shared/modules

3 - Modify App_Start/BundleConfig.cs to include State Template css and JavaScript files

4 - Modify Views/Shared/_Layout.cshtml to render state template css and JavaScript

5 - Add appropriate header and footer cshtml modules to _Layout.cshtml

6 - Refference http://beta.template.webtools.ca.gov/ for working examples
